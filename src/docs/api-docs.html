API Documentation - Biblioteca

Libros

GET /books

Obtiene todos los libros disponibles.

Respuestas: 200 Lista de libros 204 No books found
GET /books/{id}

Obtiene un libro específico por su ID.

Parámetros: id (path)
Respuestas: 200 Libro encontrado, 204 Book not found
POST /books

Crea un nuevo libro.

{
  "title": "string",
  "author": "string",
  "publisher": "string",
  "publishedIn": "number",
  "category": "string"
}
Respuestas: 201 Book created successfully, 400 Error creating book
PUT /books/{id}

Actualiza un libro existente.

Parámetros: id (path) 
{
  "title": "string (opcional)",
  "author": "string (opcional)",
  "publisher": "string (opcional)",
  "publishedIn": "number (opcional)",
  "category": "string (opcional)"
}
Respuestas: 200 Book updated successfully, 204 Book not found, 400 Error updating book
DELETE /books/{id}

Elimina un libro existente.

Parámetros: id (path)
Respuestas: 200 Book deleted successfully, 204 Book not found, 400 Error deleting book

Préstamos

POST /loans

Registra un nuevo préstamo.

{
  "inventoryNumber": "string",
  "userId": "number"
}
Respuestas: 201 Loan created successfully, 400 Required data is missing
PUT /loans/return

Registra la devolución de un libro.

{
  "inventoryNumber": "string",
  "userId": "number"
}
Respuestas: 200 Book returned successfully, 400 Required data is missing
